# YouTube Not Interested Extension

Chrome extension for quickly managing YouTube recommendations with keyboard shortcuts.

## Usage

Hover over a video on the YouTube home page and press:
- `x` - Mark as "Not Interested"
- `w` - Add to Watch Later

## Installation

1. Clone this repo
2. Go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder
